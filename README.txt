# ChooseCourseSystem
java study and only use java
纯java做的demo，供学习用
4个实体类：
    管理员：管理学生和课程，对学生和课程能够增删单条记录，修改信息，查看单条和所有信息，查看课程备注信息（选课的所有学生信息），还有修改密码，注册管理员
    学生：登陆后可以选课，取消选课，查看已选，查看未选，修改密码
    课程：管理员进行管理
    学生-选课：对应学生和课程关系，多对多
    
4个实体类的信息通过io流序列化和反序列化到对应的文件来模仿数据库操作
    比如objecOutputStream.read("学生.txt");
